# 💼 HR Management System (.NET)

## 📌 Overview
This project is a **Human Resource Management System** developed as a university assignment.  
It is designed to support basic HR operations such as employee management, department management, and policy tracking.

The system is built using **C# and .NET Framework**, with a **SQL Server** database and a simple user interface.

---

## 🎯 Objectives
- Build a management system for handling employee data
- Practice working with **.NET (C#)** and **SQL Server**
- Understand system design through **ERD, BPM, and BFD diagrams**
- Simulate real-world HR processes in a software environment

---

## ⚙️ Features

### 🔐 Authentication
- User login system
- Change password functionality
- Role-based access control (Admin/User)

### 👨‍💼 Employee Management
- Add / Edit / Delete employee records
- Manage employee profiles and personal information

### 🏢 Department Management
- Manage departments and organizational structure

### 🧾 Policy Management
- Insurance management
- Maternity policy tracking

### 🔍 Search System
- Search employee by:
  - Name
  - Employee ID
  - ID number

---

## 🛠️ Technologies Used

- **Programming Language:** C#
- **Framework:** .NET Framework
- **Database:** SQL Server
- **Frontend:** HTML / CSS
- **IDE:** Microsoft Visual Studio
- **Database Tool:** SQL Server Management Studio
